background image can be given in any tag of html

To add a background image on an HTML element, use the HTML style attribute and the CSS background-image property You can specify background images for any visible HTML element. In this example, the background image is specified for a p element. By default, the background-image will repeat itself in the direction(s) where it is smaller than the element where it is specified.
